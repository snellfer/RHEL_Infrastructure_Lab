# RHEL_Infrastructure_Lab
This project implements a professional 5-node 3-tier architecture on AlmaLinux, managed through an Ansible control node to automate infrastructure deployment. The cluster consists of two web application nodes, a database server also acting as a centralized NFS storage hub, and a dedicated load balancer to handle incoming traffic. By utilizing targeted playbooks for host resolution, NFS export configuration, and web server mounting, this setup achieves full content synchronization; any changes made to the central storage on the database node are reflected instantly across the web tier. This laboratory environment demonstrates high-level skills in Infrastructure as Code, service orchestration, and the management of distributed systems common in enterprise IT environments.
